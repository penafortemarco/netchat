1. Create a server instance typing:
    python3 server.py

2. Create a client running:
    python3 client.py

3. Digit your IP address and Port to communicate. e.g.
   127.0.0.1
   8081

4. Enjoy opening multiple clients with different IPs 
 on the same LAN:
   python3 client.py
   127.0.0.2
   8082
   *chatting....
   python3 client.py
   127.0.0.3
   8083
   *chatting....
   .....
